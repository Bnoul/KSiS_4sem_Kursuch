Вся сетевая часть в КР настроенна под виртуальной лакальную сеть. 
Чтобы настроить её на другую сеть нужно поменять значения порта и Ip на котором слушает http сервера, 
и ввести эти значения непосредственно в программе, после этого сетевая часть программы будет работать как задуманно.
IP и Port для HTTP сервера можно поменять в проекте Server_racer.
